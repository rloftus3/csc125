Valid Commands: What it does:
INSERT value, stringAdds string to the tree at the places determined by value. Nothing needs to be displayed. The value is an integer after the space after INSERT and the string is after ther comma until the end of the line.
PREORDERdisplays the contents of entire tree across the screen as a table with two rows using a preorder traversal. The first row contains the value and the second row contains the strings. (You'll likely need separate routines to print each row)
INORDERdisplays the contents of entire tree across the screen as a table with two rows using an inorder traversal. The first row contains the value and the second row contains the strings. (You'll likely need separate routines to print each row)
POSTORDERdisplays the contents of entire tree across the screen as a table with two rows using an postorder traversal. The first row contains the value and the second row contains the strings. (You'll likely need separate routines to print each row)
Any other commands will display that an invalid command was entered, the command will be displayed and exit the program.
